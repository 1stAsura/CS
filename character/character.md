## character set
 - 문자 집합
 - 문자(character)를 모아둔 것
 - ex) 라틴 문자

## Coded Character set, CSS
  - 문자와 문자에 매핑된 코드 포인트를 모아 놓은 집합
---

## Code Point
  - 코드 포인트
  - 문자 하나에 정수 하나를 매핑해둠
---

## character encoding
  - 문자 인코딩
  - 문자 집합을 메모리에 저장, 통신하기 위해 부호화하는 방식
  - ex) 모스 부호
